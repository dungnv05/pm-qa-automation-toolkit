# Architecture Decision Records (ADR)

This folder records high-value architectural decisions.

## Rules

- Each decision gets a new file named `NNNN-slug.md`.
- Numbering starts at `0001` and increments sequentially.
- Attendee approval is assumed; focus on reasoning, trade-offs, and consequences.
- Statuses:
  - `accepted` — implemented
  - `deprecated` — no longer recommended
  - `superseded` — replaced by a later ADR
