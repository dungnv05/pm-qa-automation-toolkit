# Acceptance Criteria Prompts

Use these with Notion AI or any LLM to generate acceptance criteria faster.

## Prompt Pack
- Write acceptance criteria for a user story in `[Given/When/Then` format.
- Expand a one-line story into 3-5 testable acceptance criteria.
- Identify missing edge cases for: login, checkout, API call.
- Map acceptance criteria to manual vs automation test types.
